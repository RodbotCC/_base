# Open Questions — The Master List

**Purpose:** A single consolidated place for every unanswered question, unfolded thread, or pending structural move across all 34 project files. When Jake feels stuck or has open time, this is the file to open — pick any cluster, pick any question, work on it.

**Meta-rules:**
- This file should never be "finished." New questions get added; resolved ones get moved to the bottom in the **Closed** section with a pointer to the entry that resolved them.
- Questions are clustered by theme, not by the file they originated in. Cross-references to originating files are noted inline.
- The `_` prefix on the filename keeps it at the top of an alphabetized directory listing.
- Filename starts with underscore so tools that sort alphabetically will pin it.
- Dupes have been aggregated. When three files asked the same question, the question appears once with all three origins noted.
- This doc is collaborative infrastructure — Jake or any AI instance can update it independently.
